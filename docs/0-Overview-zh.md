## 集成SDK

在调用接口时，推荐在项目中集成 SDK 的方式进行接入。通过使用 SDK，不仅可以简化开发流程、加快功能集成速度，还能有效降低后期的维护成本。火山引擎 SDK 的集成主要包括以下三个步骤：引入 SDK、配置访问凭证，以及编写接口调用代码。

## 环境要求

1. Python环境版本>=2.7
2. 如使用方舟服务(volcenginesdkarkruntime)，需要使用 >=3.6 的 Python 环境
